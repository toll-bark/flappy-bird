# flappy-bird

My first project as part of the [20 Games Challenge](https://20_games_challenge.gitlab.io/). Learning Unity. This project is best navigated through my Issues panel as I am trying to tackle this in an Agile-like fashion (despite having all my requirements up front... so it's like Agile and Waterfall?) I'm using this one to get a feel for Unity Editor and in the future hopefully can consolidate some of the steps.
